Sobre esta Aplicação

O que é?

Este portfolio digital foi desenvolvido no âmbito da unidade curricular de Programação Web da Licenciatura em Engenharia Informática da Universidade Lusófona. Tem como objetivo apresentar o percurso académico, projetos, competências e formações do aluno André Vieira (nº 21901850).

Tecnologias Utilizadas

  • Python 3.12 — linguagem principal
  • Django 6.0 — framework web (MVT)
  • SQLite — base de dados
  • HTML5 / CSS3 — templates e estilos
  • GitHub / GitHub Codespaces — controlo de versões e ambiente de desenvolvimento
  • Pillow — processamento de imagens

Funcionalidades

  • Listagem de Unidades Curriculares, Projetos e Tecnologias
  • Formulários CRUD para Projetos, Tecnologias, Competências e Formações
  • Área de administração Django
  • Upload de imagens e ficheiros
  • Registo de Making Of com erros e decisões do percurso

Como foi desenvolvido?

O desenvolvimento seguiu a arquitetura MVT (Model-View-Template) do Django. Começou pela definição dos modelos de dados, seguido da configuração do admin, depois as views e por fim os templates. O código foi escrito e testado em GitHub Codespaces, com commits regulares para o repositório.
